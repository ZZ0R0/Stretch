# Protocoles d’évaluation — V3

## 1. Objet

Mesurer ce que la V3 apporte réellement par rapport à la V2.

La V3 doit être évaluée comme une transition qualitative, pas comme une simple variation paramétrique.

---

## 2. Axes d’évaluation

### A. Dynamique
- activité auto-entretenue
- contraste spatial
- oscillations émergentes

### B. Plasticité
- apprentissage causal
- directionnalité
- sélectivité mémoire

### C. Contrôle
- réduction du rôle direct du PID
- maintien de la stabilité

---

## 3. Évaluations avant/après

## 3.1 Comparaison V2 vs V3

Mesurer :
- homogénéité spatiale ;
- fraction d’arêtes consolidées ;
- stabilité activité ;
- motifs localisés ;
- capacité d’apprentissage séquentiel.

Attendu :
- baisse de l’homogénéité ;
- baisse de la consolidation de masse ;
- apparition d’apprentissage directionnel.

---

## 3.2 Évaluation de compétition locale

Procédure :
- injecter deux stimuli concurrents ;
- observer coexistence ou élimination ;
- mesurer la dominance d’un pattern.

Mesures :
- ratio winner-take-all ;
- durée de compétition ;
- taille du motif gagnant.

---

## 3.3 Évaluation séquentielle

Procédure :
- entraîner A→B ;
- entraîner B→C ;
- tester rappel partiel A ;
- observer si A entraîne B puis C.

Mesures :
- taux de succès séquentiel ;
- différence A→B vs B→A ;
- délai de transition.

---

## 4. Métriques principales

- énergie totale ;
- énergie excitatrice ;
- énergie inhibitrice ;
- entropie de l’activité ;
- indice de contraste spatial ;
- indice de sélectivité mémoire ;
- ratio de directionnalité STDP ;
- fréquence dominante des oscillations ;
- temps de survie d’un motif.
